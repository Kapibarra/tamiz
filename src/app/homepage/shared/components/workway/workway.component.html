Тамиз – комплексное воздействие для нормализации работы кишечника​

Быстрая помощь

  • Спазмолитическое действие – снятие спазмов кишечника, сфинктеров желчевыводящих протоков, поджелудочной железы (Ферула + Айован + Гардения)
  • Болеутоляющее действие ​ (Гардения + Имбирь)
  • Ветрогонное действие (Ферула + Айован + Гардения)​

Санация ЖКТ

  • Антигельминтное действие​(Ферула + Айован + Гардения + Имбирь)​
  • Антибактериальное действие ​ (все 5 трав)
  • Противогрибковое (Айован + Имбирь)​
  • Противовирусное (Имбирь)​

Нормализация работы ЖКТ

  • Устранение холестаза – желчегонное действие (Ферула+Айован)
  • Уменьшение воспаления: ​ отека и проницаемости тканей, снижение повреждения (Гардения + Имбирь + Перец)
  • Защита тканей – гастропротективное действие ​(Перец), детоксикация и улучшение кровоснабжения ​(Перец + Имбирь), защита от окислительного стресса – антиоксидантное действие (Ферула + Гардения + Имбирь + Перец)​​

Дополнительные действия

  • Гепатопротекторное (Айован)
  • Гиполипидемическое ​ (Айован)​
  • Противоязвенное (Перец)​​
  • Противорвотное (Имбирь)​
